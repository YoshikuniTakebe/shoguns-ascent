# Shogun's Ascent

A strategic board game set in feudal Japan. Command your clan, form alliances, betray enemies, and ascend to become Shogun.

## Quick Start

```bash
npm install
npm run dev        # Frontend dev server
npm run server     # WebSocket server (port 3001)
npm run build      # Production build
```

## Features

- Hotseat and Online multiplayer modes
- 5 Clans: Iron Crane, Shadow Fox, Crimson Bear, Storm Dragon, Jade Turtle
- Season system (Spring, Summer, Autumn) with Tea, Politics, War, Cleanup phases
- Mandates: Recruit, March, Train, Harvest, Betray
- Coin-bidding combat, Monster/Oni recruitment, War Poems
- 8 Regions: Hokkaido, Oshu, Edo, Kanto, Kansai, Nagato, Shikoku, Kyushu
